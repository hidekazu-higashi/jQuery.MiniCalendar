# jQuery.MiniCalendar

簡易的なカレンダーを表示するjQueryプラグインです。

## 使い方

```html
<div id="mini-calendar"></div>
<script src="http://ajax.googleapis.com/ajax/libs/jquery/1.12.3/jquery.min.js"></script>
<script src="../jquery.minicalendar.js"></script>
```

```javascript
(function($) {
	$(function() {
		$('#mini-calendar').miniCalendar();
	});
})(jQuery);
```
